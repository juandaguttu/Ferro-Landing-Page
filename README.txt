* Juan Diego Aguilar R#11849840
* "THE FERRO" is a realistic and professional, online storefront dedicated to the weapon, Ferro, found in ARC Raiders. The idea was to create a webpage based off real gun store pages using the in-game owner, Tian Wen, who players can buy different guns or accessories within ARC Raiders.
* Features Implemented are:
* Navigational links that redirect you to the proper section
* Descriptions and Figures sourced directly from the video game
* Smooth CSS design animations

* Fonts And Color Palette:
* Montserrat (Headings) - Google Fonts
* Open Sans (Body Text) - Google Fonts
* #ffd700 (Gold - Used for buttons, highlights, and hover effects)
* Background Colors:
* #000000 (Black)
* #181818 (Dark gray for product/feature cards)
* Text Colors:
* #ffffff (White)
* #cccccc (Light gray for secondary text)
* #aaaaaa (Medium gray for footer text)
